# 🔀 Word Scramble Game (Python)

A fun and interactive **terminal-based word scramble game** built using Python.
Test your vocabulary and guessing skills by solving scrambled words!

---

## 🚀 Features

* 🎯 Random word selection
* 🔀 Smart word scrambling (ensures it's actually shuffled)
* 💡 Hint system (reveals first letter)
* ❌ Wrong attempt handling
* 🔢 Attempt counter
* 🚪 Option to quit anytime

---

## 🛠️ Tech Used

* Python (Core)
* `random` module

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/word-scramble-game.git
```

2. Navigate to the folder:

```bash
cd word-scramble-game
```

3. Run the game:

```bash
python scramble.py
```

---

## 🎮 How to Play

* Enter words separated by commas
  Example: `apple, banana, mango`

* Guess the scrambled word shown on screen

* Commands:

  * `hint` → shows first letter
  * `quit` → exits the game

---

## 📸 Sample Gameplay

```
🔀 Scrambled word: PPAEL
(5 letters)

Your guess: apple
🎉 CORRECT! WELL DONE!
```
